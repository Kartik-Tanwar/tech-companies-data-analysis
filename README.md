# Tech Companies Data Analysis

## Overview
This project analyzes global technology companies based on revenue, profit, and employees.

## Key Analysis
- Top companies by revenue and profit
- Country-wise distribution
- Employee vs revenue relationship
- Profit margin (efficiency analysis)

## Key Insights
- High revenue does not always mean high profit
- Some companies achieve higher efficiency with fewer employees
- Tech industry is concentrated in specific countries

## Tools Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Open the notebook
2. Run all cells
